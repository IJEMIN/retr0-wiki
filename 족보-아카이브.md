# 아카이브 목적
족보를 죄다 긁어 공개하는 것은, 실력없는 친목질 세력을 걷어내고 누구나 투명하게 실력을 측정받을 기회를 줍니다.
또한 일부 게으른 수업들이 변할 수 있게 압박합니다.

- 정보 비대칭을 해소합니다
- 단순 암기식 게으른 시험 방식을 변화시킬 수 있습니다
- 비공개 족보는 오픈소스 사상에 반대 됩니다
